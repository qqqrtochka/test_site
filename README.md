# Alex Web Studio — Static Frontend

Коротко: улучшённая версия одностраничного сайта-портфолио/лендинга.

Что сделано
- Доступная и адаптивная верстка (HTML/CSS).
- Улучшённый `script.js`:
  - Доступное мобильное меню с фокус-трепом
  - Reveal-анимации через IntersectionObserver
  - Аккордеон FAQ с ARIA
  - Формы: валидация, асинхронная отправка, локальное резервирование при оффлайне
- Несколько правок CSS: focus-visible, visually-hidden и переменные.

Как развернуть
1. Это статический сайт — откройте `index.html` в браузере.
2. Для production-окружения рекомендуется развернуть на `Netlify`, `Vercel` или любом статическом хостинге.

Замены и интеграция
- В `script.js` заменить `ENDPOINT` внутри `sendLead` на ваш реальный endpoint API.

Дальше (рекомендации)
- Добавить сборку на основе Vite + TypeScript + ESLint для удобства разработки.
- Настроить реальный бекенд или форму на сервисе (Formspree/Netlify Forms) для приёма лидов.
- Запустить Lighthouse и поработать над Core Web Vitals (критично для SEO).
